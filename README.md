# Display Mail User Agent T

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](./LICENSE)

Add-on for Thunderbird that displays information about the mail user agent (i.e. email software) with which the selected mail was composed.

## History

The project originated on 2005-03-31, when [Christian Weiske](https://cweiske.de/aboutme.htm) published it as "dispMUA".
In late 2007, he [handed over](https://cweiske.de/misc_extensions.htm#dispMUA) maintenance to [Jürgen Ernst](http://www.juergen-ernst.de/addons/dispmua.html), under whose stewardship it was published as [Display Mail User Agent (DispMUA)](https://addons.thunderbird.net/thunderbird/addon/display-mail-user-agent/) in the Thunderbird add-on repository for many years.
When development eventually stalled and the add-on became defunct, [Toshi](https://addons.thunderbird.net/en-US/thunderbird/user/Toshi_/) forked it in 2019 and relaunched it as "Display Mail User Agent T" — the 'T' being a nod to its new maintainer.

In July 2025, after a user [noted in a review](https://addons.thunderbird.net/en-US/thunderbird/addon/display-mail-user-agent-t/reviews/1175949/) that Toshi had retired from Thunderbird extension development, I reached out to him. Following several months of email exchange, this repository was created to carry the project forward.
