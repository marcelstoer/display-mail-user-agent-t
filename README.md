# Display Mail User Agent T

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](./LICENSE)

Add-on for Thunderbird that displays information about the mail user agent (i.e. email software) with which the selected mail was composed.

## History
This project goes all the way back to 2005-03-31 when [Christian Weiske](https://cweiske.de/aboutme.htm) first published it as "dispMUA".
At the end of 2007, Christian Weiske [handed over](](https://cweiske.de/misc_extensions.htm#dispMUA)) maintenance to [Jürgen Ernst](http://www.juergen-ernst.de/addons/dispmua.html).
For many years the add-on was available as [Display Mail User Agent (DispMUA)](https://addons.thunderbird.net/thunderbird/addon/display-mail-user-agent/) in the Thunderbird add-on repository.
At some point Jürgen Ernst stopped development and it became defunct. 
In 2019 [Toshi](https://addons.thunderbird.net/en-US/thunderbird/user/Toshi_/) forked the project and relauched it as "Display Mail User Agent T".
The 'T' suffix in the name refers to Toshi.
